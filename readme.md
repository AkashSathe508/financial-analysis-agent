# Financial Analysis Agent Crew

A multi-agent financial analysis system built with LangGraph and LangChain.

## Features

- Supervisor Agent
- Entity Extraction Agent
- Ticker Resolver Agent
- Market Agent
- Google Gemini Integration
- LangGraph Orchestration

## Tech Stack

- Python
- LangGraph
- LangChain
- Google Gemini
- yfinance
